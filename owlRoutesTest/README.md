# OWL Javascript Project Starter

This repo is a simple example of single page designs using OWL.

## Features

- [OWL](https://github.com/odoo/owl) v1.4.5
- Javascript
- Livereload
- Rollup.js v2.57
- Tests with Jest v27

## Installation

Install dependencies:

```bash
npm install
```

Dev with livereload:

```bash
npm run dev
```
